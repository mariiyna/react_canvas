# Приложение для рисования

## Функционал
- Рисование кистью 
- Выбор цвета и толщины кисти
- Ластик
- Навигация по истории (до 50 шагов):
  - горячие клавиши
  - кнопки интерфейса
- Сохранение рисунка в png
- Загрузка изображения на холст
- Рисование на сенсорных экранах

## Стек
- React
- TypeScript
- Redux Toolkit
- Tailwind CSS
